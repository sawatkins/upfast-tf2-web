Welcome! upfast.tf is a personal project offering free Team Fortress 2 servers that are easy to set up and use.

Servers are hosted on AWS EC2 instances created with Terraform and configured with Ansible.

If you're interested in playing, you can sometimes find it live during weekends.

Future Plans

My future plans for this site are to allow anyone to create and use servers for free.
