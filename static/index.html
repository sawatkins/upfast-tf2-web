about

upfast.tf is a personal project offering free Team Fortress 2 servers that are easy to set up and use. If you're interested in playing, you can sometimes find it live during weekends.

more info

How do I connect to the server?
You can connect to the server using the "Connect" button on the server table.
You can also copy it into the console with connect ip:27015

When is the server online?
The server is typically online during weekends.

Future Plans?
My future plans for this site are to allow anyone to create and use servers for free.
